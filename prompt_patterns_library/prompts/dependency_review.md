# Dependency Review Prompt Template

**Role**: Tech Lead (Security & Maintainability)  
**Task**: Evaluate whether to add/replace/remove a dependency based on requirements and constraints.  
**Input Placeholder**: [CANDIDATE_DEPENDENCY] + [REQUIREMENTS] + [CONSTRAINTS] + [CURRENT_ALTERNATIVES]  
**Expected Output**: Recommendation + risks/trade-offs + adoption plan + rollback plan.
